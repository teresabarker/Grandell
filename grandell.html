Grandell

Website built using JavaScript, HTML, and CSS

Design and Development by Teresa Barker

Behind the Name

Grandell is the website that you are on right now. The name comes from my older brother - when we were kids and playing Animal Crossing, I needed to name my town. I spent all day trying to think of a good name, only to come up short. My brother saved the day when out of the blue he goes, “What about Grandel?” Immediately I liked it. I also decided it was missing another “L”, so thus was born, Grandell.

It’s what I’ve named every new Animal Crossing town and island that I’ve built from scratch in the subsequent games, and so when choosing a name for my website, the name I used to build something from the ground up felt fitting.

Technical Details

Hosted through SiteGround, from there I also setup the domain name and manage things like the dynamic cache through the cpanel. Codebase of JavaScript, HTML, and CSS lives up in a repo on my GitHub.

Wireframes

Designing the Desktop and Mobile Layout

I used Figma to create the wireframes for Grandell.

I had an idea of what I wanted my site to look like from previous iterations of it and elements I saw on other sites with elegant and clean UI/UX, and so I started there.

The difference between the original site I envisioned in my mind to the one drafted up with images and text boxes was similar yet drastically different. Along the way, many updated and enhancements were made to the layout, color palette, fonts, etc. And many more iterations have happened along the way as I’d bounce back and forth between design and development.
